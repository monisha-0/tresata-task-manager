# Task Management Application
A mobile-first Task Management (Todo) application built using **React**, **TypeScript**, and **Material UI**, following the provided Figma design.
The application allows users to create, update, delete, and manage tasks grouped by their status, with a clean and intuitive user experience.

## Features

### Core Functionality
- Create new tasks with title, description, and status
- Edit existing tasks with pre-populated form values
- Delete tasks
- Mark and manage task status (Pending, In Progress, Completed)
- Tasks grouped by status using accordion layout
- Mobile-first UI aligned with the provided Figma design

### Enhanced UX Features
- Status-based color indicators (consistent across list and edit views)
- Accordion-based task grouping for better readability
- Hover-based action visibility (Edit / Delete)
- Cancel actions reset form state to avoid stale data
- Client-side validation for required fields

## How to Run Locally

### Prerequisites
- Node.js (v18 or later recommended)
- npm or yarn

### Steps

```bash
# Clone the repository
git clone - https://github.com/monisha-0/tresata-task-manager.git

# Navigate to the project directory
cd tresata-task-manager

# Install dependencies
npm install

# Start development server
npm run dev
